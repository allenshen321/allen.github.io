'正则表达式总结'

发表于 2018-08-28 | 分类于 学习笔记

1、正则表达式

正则表达式，用来检索、替换那些符合某个规则的文本。正则表达式是对字符串操作的一种逻辑公式，就是用实现定义好的一些特殊字符、及这些特定字符的组合，组成一个“规则字符串”，这个“规则字符串”用来表达对字符串的一种过滤逻辑。

正则表达式匹配逻辑：

首先有正则表达式引擎将表达式字符串转化为正则表达式，这个表达式包含着匹配信息的规则。其次，正则表达式对象对文本进行匹配后得到结果。

2、re模块使用步骤：

  1. 使用compile()函数将正则表达式的字符串编译为pattern对象；
  2. 通过pattern对象提供的一系列方法对文本进行匹配查找，获得匹配结果，一个Match对象；
  3. 最后使用Match对象提供的属性和方法获得信息，根据需要进行其他的操作。

3、re模块中的函数用法：

（1）compile函数

compile函数用于编译正则表达式，生成一个Pattern对象，使用方法如下：

import re
pattern = re.compile(r'\d+')

Pattern对象一些常用的方法：

  • match方法：从起始位置开始查找，一次匹配
  • search方法：从任何位置开始查找，一次匹配；
  • findall方法：全部匹配，返回列表；
  • finditer方法： 全部匹配，返回迭代器；
  • split方法： 分割字符串，返回列表；
  • sub方法：替换

Pattern对象方法介绍

1、match方法

match方法用于查找字符串的头部，是一次匹配，只要找到了一个匹配的结果就返回，而不是查找所有的匹配结果。使用形式是：

match（string[,pos[,endpos]])
其中，string是待匹配的字符串，pos和endpos是可选参数，指定的字符串的起点和终点的位置，默认是0和len(字符串长度).因此，当不指定pos和endpos时，match方法默认匹配字符串的头部。
当匹配成功时，返回一个Match对象，如果没有匹配上，则返回None。

import re
pattern = re.compile(r'\d+')  # 用于匹配至少一个数字，将字符长编译成pattern对象
m = pattern.match('one12ksoenndsk')  # match方法，查找字符串头部
print(m)  # 打印结果为None
# 指定查找位置
m = pattern.match('one12twothree34four', 2, 10) # 查找2到10的位置范围匹配
print(m)  # 打印结果为None

m = pattern.match('one12twothree34four', 3, 10) # 查找位置3到10
print(m)  # 打印结果：<_src.SRE_Match object at 0x10a42aac0>
# group可以查看匹配内容
m.group(0)  # 可省略0

上面，但匹配成功时会返回一个Match对象，其中：

  • group([group1, …])方法用于获得一个或多个分组匹配的字符串，直接使用group()或group(0),可直接获得整个匹配的子串。
  • start([group])方法用于获得分组匹配的自创在整个字符串中的起始位置，参数默认值为0；
  • end([group])方法用于获得分组匹配的子串在整个字符串中的结束位置；
  • span([group])方法返回(start(group),end(group))。
2、search方法

search方法用于查找字符串的任何位置，它也是一次匹配，只要找到一个匹配的结果就返回，它的使用形式是：

search(string[, pos[, endpos]])

其中，string是待匹配的字符串，pos和endpos是可选参数，指定字符串的起始和终点位置，默认值是0和len(字符串长度)。

如果成功匹配，返回 一个Match对象，否则返回None。

eg:
import re
pattern = re.compile('\d+') 
m = pattern.search('one12twothree34four') # 这里如果使用match方法则不匹配。
m.group()  # 结果是'12'
3、findall方法

findall方法匹配整个字符串，所得所有的结果；

findall方法使用形式如下：

findall(string[, pos[, endpos]])

其中，string是待匹配字符串，pos和endpos是可选参数，指定字符串的起始和终点位置，默认值为0和len(字符串长度).

findall以列表的形式返回全部匹配的子串，如果没有匹配，则返回一个空列表。

import re
pattern = re.compile(r'\d+')  # 查找数字
result1 = pattern.findall('hello 1234 2283')
result2 = pattern.findall('one12two1three232four35', 0, 10)
print result1  # 返回结果是['1234', '2283']
print result2  # 打印结果为['12', '1']
4、finditer方法

finditer方法的行为和findall的行为类似，也是搜索整个字符串，获得所有的匹配结果。当返回一个顺序访问每一个匹配结果（Match对象)的迭代器。可使用for遍历每个元素。

5、sub方法

sub方法用于替换。使用形式：

sub(repl, string[, count])

其中，repl可以是字符串也可以是一个函数：

  • 如果repl是字符串，则会使用repl替换字符串中的每一个匹配的字符串，并返回替换后的字符串，另外，repl还可以使用id的形式来引用分组，但不能使用编号0；
  • 如果repl是函数，这个方法应当只接受一个参数（Match对象），并返回一个字符串用于替换（返回的字符串中不能在引用分组）。

  • count用于指定最多替换次数，默认是全部替换。

    import re
    p = re.compile(r’(\w+) (\w+)’) # \w = [A-Za-z0-9]
    s = ‘hello 123, hello 456’
    print(p.sub(r’hello world’, s) # 使用’hello world’ 替换 ‘hello123’和’hello 456’ 打印结果是’hello world, hello world’

匹配中文

中文的unicode编码范围在[u4e00-u9fa5],但是这其中没有包括全角（中文）标点，在大部分情况下是够用的。

import re
title = u'你好，hello，世界'
pattern = re.compile(ur'[\u4e00-u9fa5]+')
result = pattern.findall(title)
print result  # 打印结果是[u'\u4f60\u597d', u'\u4e16\u754c']

贪婪模式和非贪婪模式

  1. 贪婪模式：在整个表达式匹配成功的前提下，尽可能多的匹配（*）；
  2. 非贪婪模式：在整个表达式匹配成功的前提下，尽可能少的匹配(?);
  3. Python里数量词默认是贪婪地。

示例：

import re 
pattern = re.compile(r'ab*')
m = pattern.findall('abbbbc')  # 贪婪模式，匹配就可能多的b，结果是abbbb
pattern1 = re.compile(r'ab*?')
m1 = pattern1.findall('abbbbc') # 非贪婪模式，匹配尽可能少匹配b，所以没有b，结果是 a
