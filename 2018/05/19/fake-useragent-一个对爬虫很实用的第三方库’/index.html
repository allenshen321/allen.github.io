fake-useragent,一个对爬虫很实用的第三方库

发表于 2018-05-19 | 分类于 爬虫库

前言

最近发现一个特别简单实用的python第三方库，fake-useragent库，它能提供request请求头中的User-Agent信息。

使用

1、安装fake-useragent库

pip install fake-useragent

2、使用实例，直接上代码
from fake_useragent import UserAgent
ua = UserAgent()

3、值得注意的几点

1.fake-useragent会在本地系统的临时文件里面储存UserAgent数据，比如/tmp文件夹中。

  • 如果想更新以保存的本地数据，可以用下面的方法：

    from fake_useragent import UserAgent
ua = UserAgent()
ua.update()

  • 如果不想缓存本地数据文件，可以在建立UserAgent实例的时候设置参数’cache=False’

    from fake_useragent import UserAgent
ua = UserAgent(cache=False)
