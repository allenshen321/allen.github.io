flask web表单

发表于 2018-09-08 | 分类于 flask

第三方扩展

尽管flask的请求对象提供的信息足够用力处理web表单,但是比较繁琐,重复操作.flask-wtf扩展可以使处理web表单的过程变得更方便简洁.

flask-wtf扩展,将WTForm包进行了包装,方便集成到flask程序中.

安装方式:

pip install flask-wtf

csrf(跨站请求伪造)保护

为了实现csrf保护,flask-wtf需要程序设置一个秘钥.flask-wtf使用这个秘钥生成加密令牌,再用凌攀验证请求中表单数据的真伪.

程序设置秘钥的方法:

from flask import Flask

app = Flask(__name__)
app.config['SECRET_KEY'] = '一个很难猜的字符串'
# 为了安全行,秘钥不应该写到代码里,而是要保存到环境变量中.

app.config字典用来存储框架,扩展和程序本身的配置变量.

SECRET_KEY配置变量是通用秘钥,可以在flask和多个第三方扩展中使用.

表单类

使用flask-wtf时,每个表单都继承Form的类表示,该类定义表单中的一组字段,每个字段都用对象表示.字段对象可以添加一个或多个验证函数,验证函数用来验证用户输入的值是否符合要求.

实例：定义一个文本字段和一个提交按钮．
from flask.ext.wtf import Form
from wrforms import StringField, SubmitField
from wtform.validators import Required

class NameForm(Form):
    name = StringField('your name', validators=[Required()])
    submit = SubmitField('Submit')
flask-wtf支持的字段和验证函数
  1. flask-wtf支持的HTML标准字段
字段类型 说明
StringField 文本字段
TextAreaField 多行文本字段
PasswordField 密码文本字段
HiddenField 隐藏文本字段
DateField 文本字段,值为datetime.date格式
DateTimeField 文本字段,值为datetime.datetime格式
IntegerField 文本字段,值为整数
DecimalField 文本字段,值为decimal.Decimal
FloatField 文本字段,值为浮点数
BooleanField 复选框,值为True或False
RadioField 一组单选框
SelectField 下拉列表
SelectMultipleField 下拉列表,可选择多个列表
FileField 文件上传字段
SubmitField 表单提交按钮
FormField 把表单作为字段嵌入另一个表单
FieldList 一组指定类型的字段
  1. flask-wtf 支持的验证函数
验证函数 说明
Email 验证电子邮件地址
EqualTo 比较两个字段的值,常用语要求输入两次密码进行确认的情况
IPAddress 验证IPv4网络地址
Length 验证输入字符串的长度
NumberRange 验证输入的值在数字范围之内
Optional 无输入值时,跳过其他验证函数
Required 确保值非空
Regexp 使用正则表达式验证输入值
URL 验证URL
AnyOf 确保输入值在可选值列表中
NoneOf 确保输入值不在可选值列表中
