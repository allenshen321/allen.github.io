ubuntu 环境变量

发表于 2018-09-11 | 分类于 ubuntu

查看环境变量

三种方法查看环境变量:

  1. env

env是environment的缩写,用于列出所有的环境变量.

  1. export

单独使用export命令可以列出所有的环境变量,但是它还有其他的作用,比如,添加环境变量

  1. echo $PATH

用于列出变量PATH的值,里面包含了已添加的目录.

设置环境变量

有两种方式设置环境变量:

  1. 把路径添加到PATH中

可以直接添加环境变量的PATH中. $PATH表示变量的值,包含已有的目录.

注意

这种方法,遇到路径同名的时候,执行顺序是,那个目录先被查询,则那个目录下的命令先被执行.

示例

# 添加到PATH末尾
export PATH=$PATH:/path/to/your/dir

# 添加到PATH开头
export PATH=/path/to/your/dir:$PATH
  1. 命名一个新的环境变量

直接命名一个新的环境变量,用于其他程序的引用:

export VAR_NAME=value

作用域

环境变量的作用有以下几种:

1.当前终端

打开一个终端,添加环境变量

export CLASS_PATH=./JAVA_HOME/lib:$JAVA_HOME

终端添加的环境变量是临时的,只适用域当前终端,关闭当前终端或在另一个终端中,添加的环境变量便会无效.

2.用于当前用户

如果只需要添加环境变量对当前用户有效,可以写入用户主目录下的.bashrc文件,

vim ~/.bashrc

添加语句:

export CLASS_PATH=./JAVA_HOME/lib:$JAVA_HOME/jre/lib

注销或者重启可以使修改生效,如果要使添加的环境变量马上生效:

source ~/.bashrc
3.用于所有用户

要使环境变量对所有用户有效,可以修改profile文件

sudo vim /etc/profile

添加语句

export CLASS_PATH=./JAVA_HOME/lib:$JAVA_HOME/jrc/lib

注销或者重启可以使修改生效,如果想立马生效的方法是:

source /etc/profile
