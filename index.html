函数式编程学习

发表于 2018-09-08 | 分类于 学习笔记
1.简述函数式编程

在函数式编程中,函数是基本单位,变量只是一个名称而非存储单位.除了匿名函数外,python还使用filter(),map(),reduce(),apply()来支持函数式编程.

  1. filter()函数

用于过滤序列,过滤掉不符合条件的序列,返回由符合条件的序列组成的列表.在python3中,返回的是符合条件的元素的迭代器对象.

filter()函数接受两个参数,第一个是函数名,第二个是序列,序列的每个元素作为函数的参数进行判断,返回True或False,最后将所有结果True的元素返回.

1、生成器

生成器的创建方法：

(1)通过列表生成式创建

可以通过将列表生成式的[]改成()

eg: 
# 列表生成式
L = [ x*2 for x in range(5)]  # L = [0, 2, 4, 6, 8]
# 生成器
G = （ x*2 for x in range(5))  # 此时的G是,<generator object <genexpr> at 0x7f626c132db0>

创建列表生成式和生成器的区别只是最外层的()和[],列表生成式是一个列表，而生成器事宜个可迭代对象。生成器对象可以通过for语句遍历每个元素。

for each in G:
print(each)  # 打印结果是 0 2 4 6 8
